Cybersecurity & Data Protection

1. What is Cybersecurity?

Answer: Cybersecurity is the practice of protecting computers, networks, and data from unauthorized access, attacks, or damage.

2. Define Data Privacy.

Answer: Data Privacy refers to the protection of personal information and ensuring it is collected, stored, and shared appropriately and only with consent.

3. What is Malware?

Answer: Malware (Malicious Software) is any harmful program such as viruses, worms, trojans, spyware, and ransomware designed to damage or exploit systems.

4. What is a Computer Virus?

Answer: A virus is a malicious program that attaches to a host file and replicates itself when the infected file runs.

5. What is Ransomware?

Answer: Ransomware is malware that encrypts a user's files and demands payment (ransom) to restore access.

6. Define Phishing.

Answer: Phishing is a cyberattack where attackers pretend to be legitimate organizations to steal personal data (e.g., through fake emails or websites).

7. What is Social Engineering?

Answer: Social Engineering is manipulating people to reveal confidential information instead of hacking systems.

8. What is a Firewall?

Answer: A firewall is a security system that filters incoming and outgoing network traffic based on security rules, preventing unauthorized access.

9. What is Encryption?

Answer: Encryption is the process of converting data into unreadable form to protect it from unauthorized access.

10. Difference between Symmetric and Asymmetric Encryption.

Answer: Symmetric Encryption: Uses one key for encryption and decryption. (e.g., AES) Asymmetric Encryption: Uses two keys (public & private). (e.g., RSA)

11. What is Two-Factor Authentication (2FA)?

Answer: 2FA is a security method requiring two verification forms (e.g., password + OTP code) to confirm identity.

12. What is a Brute Force Attack?

Answer: A Brute Force Attack tries every possible password combination until it finds the correct one.

13. What is a DDoS Attack?

Answer: A Distributed Denial of Service attack overwhelms a server with massive traffic to make it unavailable.

14. Define Data Breach.

Answer: A Data Breach occurs when confidential data is accessed, stolen, or exposed without authorization.

15. What is a Strong Password?

Answer: A secure password is at least 12+ characters long and includes uppercase, lowercase, numbers, and symbols.

16. What is Backup and Why is it Important?

Answer: A backup is a copy of data stored separately so it can be restored in case of loss, corruption, or ransomware attacks.

17. What is HTTPS and Why is it Used?

Answer: HTTPS encrypts data transferred over websites, ensuring secure communication between browser and server.

18. What is Access Control?

Answer: Access Control restricts who can view or use resources in a system based on permissions.

19. What is Patch Management?

Answer: Patch Management involves updating software regularly to fix vulnerabilities and prevent attacks.
