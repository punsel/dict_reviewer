Database Concepts

1. What is a Database?

Answer: A database is an organized collection of data stored and accessed electronically.

2. What is a DBMS?

Answer: Database Management System, software that stores, manages, retrieves, and updates data in a database.
Examples: MySQL, PostgreSQL, Oracle, SQL Server.

3. What is the Relational Model?

Answer: A model where data is organized in tables (relations) consisting of rows and columns.

4. What is a Table (Relation) in a database?

Answer: A table is a structure that stores data in rows (records) and columns (attributes).

5. What is a Primary Key?

Answer: A unique identifier for each row in a table.
Example: student_id, employee_id.

6. What is a Foreign Key?

Answer: A field that links two tables by referring to the Primary Key of another table.
Used to maintain relationships.

7. What is SQL?

Answer: Structured Query Language used to store, retrieve, and manipulate data in relational databases.

8. Write an SQL query to retrieve all data from a table named Students.

SELECT * FROM Students;

9. What command is used to insert data into a table?

Answer: INSERT

INSERT INTO Students (id, name) VALUES (1, 'Juan');

10. Which SQL command is used to update data?

Answer: UPDATE

UPDATE Students SET name = 'Maria' WHERE id = 1;

11. Which SQL command deletes records from a table?

Answer: DELETE

DELETE FROM Students WHERE id = 1;

12. What is a JOIN in SQL?

Answer: A JOIN is used to combine rows from two or more tables based on a related column.

13. What is an INNER JOIN?

Answer: Returns only matching records from both tables.

SELECT s.name, c.course_name
FROM Students s
INNER JOIN Courses c ON s.course_id = c.id;

14. What is a LEFT JOIN?

Answer: Returns all records from the left table, and matching records from the right table.

15. What is Normalization?

Answer: The process of organizing data to reduce redundancy and improve efficiency.

16. What is 1st Normal Form (1NF)?

  • No repeating groups
  • Each cell contains one value (atomic values)

17. What is 2nd Normal Form (2NF)?

Answer: Must be in 1NF and no partial dependency on a composite primary key.

18. What is 3rd Normal Form (3NF)?

Answer: Must be in 2NF and no transitive dependency (non-key fields depending on other non-key fields).

19. What is a Transaction in a Database?

Answer: A transaction is a set of operations executed as a single unit where either all succeed or none occur.

20. What are the ACID Properties of Transactions?

PropertyMeaning
AtomicityAll or nothing execution
ConsistencyMaintains valid database state
IsolationTransactions do not interfere
