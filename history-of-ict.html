Fundamentals of ICT: History & Basics

History & Inventors

  • Who is known as the Father of Computers? Charles Babbage
  • What did Charles Babbage invent? Analytical Engine
  • Who invented the first mechanical calculator? Blaise Pascal
  • Who created the punched card system used in early computing? Herman Hollerith
  • Who is considered the first computer programmer? Ada Lovelace
  • ENIAC stands for? Electronic Numerical Integrator and Computer
  • What generation of computers used vacuum tubes? 1st Generation
  • What replaced vacuum tubes in the 2nd generation? Transistors
  • What innovation characterized the 3rd generation of computers? Integrated Circuits (ICs)
  • What technology characterizes the 4th generation of computers? Microprocessor

Computer Basics / Components

  • ICT stands for? Information and Communication Technology
  • What is hardware? The physical components of a computer
  • What is software? Programs and instructions the computer runs
  • Which device is used for long-term storage? Hard Disk / SSD
  • Which component executes instructions? CPU (Central Processing Unit)
  • RAM stands for? Random Access Memory
  • Is RAM volatile or non-volatile? Volatile (data is lost when power is off)
  • ROM stands for? Read Only Memory
  • Which memory contains BIOS? ROM
  • What is a peripheral device? Any device that connects to the computer externally (Mouse, Keyboard, Printer)

Binary & Number Representation

  • Binary system is base what? Base 2
  • Decimal system is base what? Base 10
  • Hexadecimal system is base what? Base 16
  • 1 Byte = how many bits? 8 bits
  • 1 KB = how many bytes? 1024 bytes
  • Convert binary 1011 to decimal. 11
  • Convert decimal 15 to binary. 1111
  • Convert hex A to decimal. 10
  • Convert decimal 255 to hex. FF
  • Which is the smallest unit of data? Bit

Software, OS, & Applications

  • Operating System is what type of software? System Software
  • Give two examples of operating systems. Windows, Linux, macOS, Android
  • What is application software? Software designed for user tasks (e.g., MS Word, Photoshop)
  • What is open-source software? Software whose code is publicly available
  • Example of open-source OS: Linux / Ubuntu
  • What is cloud computing? Accessing computing resources over the internet
  • Give an example of cloud storage. Google Drive / OneDrive / Dropbox

Networking & Internet

  • What is a network? A group of connected computers that share resources.
  • LAN stands for? Local Area Network
  • WAN stands for? Wide Area Network
  • What is the World Wide Web (WWW)? A collection of web pages accessed via the Internet.
  • Who invented the World Wide Web? Tim Berners-Lee
  • What does URL stand for? Uniform Resource Locator
  • What does ISP stand for? Internet Service Provider
  • What is an IP address used for? Identifying devices on a network.
  • Difference between HTTP and HTTPS? HTTPS is secure (encrypted).

Cybersecurity & Data Protection

  • What is a computer virus? A malicious program that spreads and damages systems.
  • What is a firewall? Security system that controls network traffic.
  • What is phishing? Tricking users to reveal personal information.
  • Strong passwords should contain what elements? Uppercase, lowercase, numbers, and symbols.
