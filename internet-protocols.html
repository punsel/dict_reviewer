Internet Protocols

1. What does IP stand for and what is its main function?

Answer: Internet Protocol; it handles addressing and routing of data between devices on a network.

2. What is the difference between IPv4 and IPv6?

Answer: IPv4: Uses 32-bit addresses (e.g., 192.168.1.1), supports about 4.3 billion addresses. IPv6: Uses 128-bit addresses (e.g., 2001:db8::1), supports virtually unlimited device addressing.

3. Which protocol is used to send webpages from a server to a browser?

Answer: HTTP (HyperText Transfer Protocol).

4. What is HTTPS and how is it different from HTTP?

Answer: HTTPS = HTTP secured with SSL/TLS encryption to protect data from being intercepted.

5. Which protocol is responsible for reliable communication and guaranteed delivery?

Answer: TCP (Transmission Control Protocol).

6. Which protocol is faster but does NOT guarantee data delivery?

Answer: UDP (User Datagram Protocol).

7. Give one example of an application that commonly uses UDP.

Answer: Online gaming, VoIP, or video streaming (where speed is needed more than reliability).

8. Which protocol translates domain names into IP addresses?

Answer: DNS (Domain Name System).

9. What is the default port of DNS?

Answer: Port 53.

10. Which protocol is used to send email from a client to a server?

Answer: SMTP (Simple Mail Transfer Protocol).

11. Which protocols are used to receive email?

Answer: POP3 (Port 110) and IMAP (Port 143).

12. What is FTP used for?

Answer: FTP (File Transfer Protocol) is used to upload and download files between computers.

13. Which secure alternative to FTP is commonly used?

Answer: SFTP (Secure File Transfer Protocol).

14. What protocol is used to remotely log into another computer?

Answer: SSH (Secure Shell).

15. Which addressing method uses MAC addresses?

Answer: Data Link Layer (Layer 2) uses MAC addresses.

16. Which addressing method uses IP addresses?

Answer: Network Layer (Layer 3).

17. What protocol is used for routing packets between networks?

Answer: BGP (Border Gateway Protocol) or OSPF/RIP (Interior routing protocols).

18. What protocol is used to automatically assign IP addresses to devices?

Answer: DHCP (Dynamic Host Configuration Protocol).

19. What is the purpose of ARP?

Answer: ARP (Address Resolution Protocol) maps IP addresses to MAC addresses.

20. What protocol checks if a device is reachable on a network (used in ping)?

Answer: ICMP (Internet Control Message Protocol).
