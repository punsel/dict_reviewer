Networking Concepts

1. What is a computer network?

Answer: A network is a group of devices connected together to share resources, data, and communication channels.

2. What does LAN stand for and what is it used for?

Answer: Local Area Network — used to connect devices in a small area like a school, office, or home.

3. What does WAN stand for?

Answer: Wide Area Network, connecting devices across large geographic distances, e.g., the Internet.

4. What is the main purpose of the OSI Model?

Answer: To provide a standard framework for understanding how data moves across networks.

5. How many layers are in the OSI Model and what are they (from top to bottom)?

Answer: 7 Layers:

  • Application
  • Presentation
  • Session
  • Transport
  • Network
  • Data Link
  • Physical

6. Which OSI layer is responsible for routing and IP addressing?

Answer: Network Layer (Layer 3)

7. Which OSI layer ensures reliable delivery and error correction?

Answer: Transport Layer (Layer 4)

8. Which OSI layer defines physical connections like cables and signals?

Answer: Physical Layer (Layer 1)

9. What are the 4 layers of the TCP/IP Model?

  • Application
  • Transport
  • Internet
  • Network Access

10. What protocol is connection-based and reliable?

Answer: TCP (Transmission Control Protocol)
Provides acknowledgment and error checking.

11. What protocol is connectionless and faster but unreliable?

Answer: UDP (User Datagram Protocol)
Used for streaming & gaming.

12. What does IP stand for and what is its purpose?

Answer: Internet Protocol — responsible for addressing and routing packets.

13. What is an IP address?

Answer: A unique numeric identifier for a device on a network.
Example IPv4: 192.168.0.1

14. What is the difference between IPv4 and IPv6?

IPv4IPv6
32-bit, ~4.3 billion addresses128-bit, almost unlimited
Example: 192.168.1.1Example: FE80::1EB2:A33C

15. What is a Subnet Mask used for?

Answer: To separate the network portion and host portion of an IP address.
Example: 255.255.255.0

16. What does DNS do?

Answer: Domain Name System converts domain names → IP addresses.
Example: google.com → 142.250.190.14

17. What is DHCP used for?

Answer: Dynamic Host Configuration Protocol automatically assigns IP addresses to devices on a network.

18. What is a MAC address?

Answer: A unique hardware address assigned to a network interface.
Example: 00:1A:2B:3C:4D:5E
Found in Layer 2: Data Link Layer

19. What does HTTP and HTTPS stand for?

Answer:

HTTP — Hypertext Transfer Protocol
HTTPS — HTTP Secure (encrypted)

20. What is a Router?

Answer: A device that forwards data between networks, using IP addresses to choose the best path.
